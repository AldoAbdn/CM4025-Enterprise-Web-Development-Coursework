API

Documentation for the backend RESTful API

Auth

Authentication api for retrieving access tokens

Login

URL /api/auth/login
Method POST
Request
Headers { 'Accept': 'application/json', 'Content-Type': 'application/json' }
Body { username: String, password: String }
Response
Headers { 'Content-Type': 'application/json' }
Payload { auth: Boolean, token: String, message: String }

Sign Up

URL /api/auth/signup
Method POST
Request
Headers { 'Accept': 'application/json', 'Content-Type': 'application/json' }
Body { username: String, password: String }
Response
Headers { 'Content-Type': 'application/json' }
Payload { auth: Boolean, token: String, message: String }

Logout

URL /api/auth/logout
Method POST
Request
Headers { 'Accept': 'application/json', 'Content-Type': 'application/json' }
Body N/A
Response
Headers { 'Content-Type': 'application/json' }
Payload { auth: Boolean, token: String, message: String }

User

Retrieving and updating user details

Get Details

URL /api/user/getDetails
Method GET
Request
Headers { 'Accept': 'application/json', 'Content-Type': 'application/json' 'x-access-token': String }

This service requires an access token (x-access-token). This can be retrieved from the Auth service
Response
Headers { 'Content-Type': 'application/json' }
Payload { username: String, feedSources: Array, login: Date, lastLogin: Date }

Add Feed

URL /api/user/addFeed
Method POST
Request
Headers { 'Accept': 'application/json', 'Content-Type': 'application/json' 'x-access-token': String }

This service requires an access token (x-access-token). This can be retrieved from the Auth service
Body { feedSource: String }

Currently supported feed source names

  • World - World United States - Goodle News
  • Defence Blog
  • BBC News - World
Response
Headers { 'Content-Type': 'application/json' }
Payload { username: String, feedSources: Array, login: Date, lastLogin: Date }

Feed

Retrieving supported RSS feeds by list of source names

Get Feeds

URL /api/feed/getFeeds?feeds=[feedSource...,]
Method GET
Request
Headers { 'Accept': 'application/json', 'Content-Type': 'application/json' 'x-access-token': String }

This service requires an access token (x-access-token). This can be retrieved from the Auth service
Params feeds: Array[String]

Supported feed source names (use exact values)

  • World - World United States - Goodle News
  • Defence Blog
  • BBC News - World
Response
Headers { 'Content-Type': 'application/json' }
Payload { sourceName: { title: String, description: String, link: String, item: [ title: String, description: String, link: String, ... ] } }

Structure can vary depending on feeds added

This service requires an access token (x-access-token). This can be retrieved from the Auth service
Response

Currently supported feed source names
